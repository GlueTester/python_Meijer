# py-meijer

Python module for interacting with Meijer's mPerks. 



## Shopping List.

    $ meijer list clear
    $ meijer list add
    Enter your shopping list. Complete with a blank line.
    - Foo
    - Bar
    - Ipsum
    - Lorem
    $ meijer list show
    [ ] Lorem
    [ ] Ipsum
    [ ] Bar
    [ ] Foo
