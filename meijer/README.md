# py-meijer

Python module for interacting with Meijer's mPerks. 



## Commandline Program ```meijer```

### Shopping List.

CLear shopping list:

    $ meijer list clear

Add items to shopping list:

    $ meijer list add
    Enter your shopping list. Complete with a blank line.
    - Foo
    - Bar
    - Ipsum
    - Lorem

Show shopping list:

    $ meijer list show
    [ ] Lorem
    [ ] Ipsum
    [ ] Bar
    [ ] Foo
